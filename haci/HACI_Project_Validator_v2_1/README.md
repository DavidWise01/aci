# HACI Project Validator v2.1

Run tests:

```bash
python test_haci_project_validator_v2_1.py
```

Validate examples:

```bash
python haci_project_validator_v2_1.py examples/clean_project --strict
python haci_project_validator_v2_1.py examples/bad_project
python haci_project_validator_v2_1.py examples/open_project --strict
```

v2.1 is a conversation-engine patch over v2.0. It does not change HACI syntax.
