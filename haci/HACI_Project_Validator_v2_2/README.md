# HACI Project Validator v2.2

Run tests:

```bash
python test_haci_project_validator_v2_2.py
```

Validate examples:

```bash
python haci_project_validator_v2_2.py examples/clean_project --strict
python haci_project_validator_v2_2.py examples/bad_project --strict
```

v2.2 is a resolution and return-pairing patch over v2.1. It does not change HACI syntax.
